
This is shor user guide for domain-specific modelling with OSMO tester

@Author Olli-Pekka@Puolitaival.fi

Osmo simple language is a modellin language for MetaEdit+ 5.0 version.
Modelling language makes easy to train modelling and learn osmo tester.
Later you can develop your own modelling language based on this simple one.

How to take in use:
- Install meta edit+ 5.0 or newer from www.metacase.com
- Make an empty project
- import language.met or language.mxt (.met has more information)
- import models.mec or models.mxm (.mec has more information)
- Generate model and run it using OSMO tester which is available in http://osmo.googlecode.com/
